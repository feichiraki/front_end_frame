#### 3、yarn指令

##### 3.1 yarn常用指令

```bash
# 1、添加依赖包
yarn add packagename	# 下载依赖包到本地
yarn add packagename1 packagename2 ...	# 下载多个依赖包到本地

# 2、查看包的版本,然后下载特定版本包
npm view 包名 versions	# 因为yarn没有查看包所有版本号的命令，所以使用npm
yarn add packagename@版本号


```
